# 🎓 Chanakya University Inquiry Portal

 **Full Stack University Inquiry Portal** developed using **React (Vite)** and **Django REST Framework**. This project allows students and parents to explore university information such as courses, faculty, hostel facilities, and transport routes, along with inquiry registration.

## 📌 Features

* 📚 Course Details Display
* 👨‍🏫 Faculty Information with Course Filter
* 🏨 Hostel Facilities & Room Availability
* 🚌 Transport Routes & Bus Information
* 📝 Inquiry Registration Popup
* 🔗 REST API Integration
* 📱 Responsive User Interface
